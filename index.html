About This Assessment

This comprehensive questionnaire will help identify potential ADHD type patterns based on Dr. Amen's framework. The assessment examines seven distinct presentations:

  • Classic ADD - Hyperactive and impulsive
  • Inattentive ADD - Quiet and unfocused
  • Over-focused ADD - Stuck in negative patterns
  • Temporal Lobe ADD - Memory and mood issues
  • Limbic ADD - Depression-like symptoms
  • Ring of Fire ADD - Extreme sensory sensitivity
  • Anxious ADD - Worry and physical tension

The assessment takes approximately 10-15 minutes to complete. Answer each question based on how frequently you experience these symptoms.
